Pixelknight

A multiplayer 2D platformer game with pixel art graphics, built with JavaScript and WebSockets.

Multiplayer

Play with friends in real-time over WebSockets

Pixel Art

Beautiful retro-style pixel art graphics

Combat

Fight enemies with your sword

Platforming

Jump between platforms and collect coins

Connect to Game Server

How to Play Multiplayer:

  1. Run the server on your computer with: node servercode.js
  2. Enter your local IP address and port above (e.g., 192.168.1.100:8080)
  3. Click "Connect to Server" to join the game

Note: If you're running the server on your local machine, use your local IP address (not "localhost") when connecting from other devices.
