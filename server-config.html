Pixelknight Server Configuration

How to Play Multiplayer:

  1. Run the server on your computer or a server with: node servercode.js
  2. Enter the server address (IP and port) in the field above
  3. Click "Connect to Server" to join the game

Note: If you're running the server on your local machine, use "localhost:8080". If connecting to someone else's server, use their IP address and port.
